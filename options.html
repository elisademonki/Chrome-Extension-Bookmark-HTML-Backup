Bookmark HTML Backup

Chrome erlaubt Extensions hier nur einen Pfad relativ zum Chrome-Downloadordner. Beispiel: bookmark-backups speichert nach Downloads\bookmark-backups\....

Ein direkter absoluter Pfad wie C:\FESTPLATTEN\... ist in einer normalen Chrome-Extension nicht erlaubt.

Standardmodus für einfachen Icon-Klick

Bedienung

  • Ein Klick auf das Extension-Icon: Backup im gespeicherten Standardmodus.
  • Doppelklick auf das Extension-Icon: Optionen öffnen.
  • Rechtsklick auf das Extension-Icon: direkt kompakt oder mit Favicons sichern.
